# Plan, compress, execute — Turn a bounded junior task into verified delivery

1. Restate the assigned task, scope, constraints, and acceptance checks in your own words.
2. Write a brief plan with the smallest ordered steps needed to deliver it.
3. Execute the plan without expanding the scope; report any blocking ambiguity instead of inventing a design.
4. Verify the result against the original task and its acceptance checks, then report the files changed and evidence.
